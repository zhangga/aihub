# Topic Boundary

Use this file to decide whether an item belongs in the report.

## Core Rule

Keep an item only if it has a clear and meaningful overlap between games and AI.

That overlap usually takes one of two forms:

- a game company, game platform, or game workflow is adopting AI
- an AI company is building a game, interactive entertainment product, or game-facing capability

## In Scope

- AI NPC systems, agent behavior, or player-facing interactive systems in games
- AIGC art, audio, voice, localization, quest, or narrative tooling used by game teams
- AI-assisted QA, live ops, moderation, support, balancing, or content pipelines for games
- game engines, game platforms, or UGC ecosystems adding AI creation workflows
- AI companies launching game, simulation, interactive media, or player-facing entertainment demos or products
- partnerships between game companies and AI vendors where the game use case is explicit
- funding, M&A, policy, labor, or platform decisions that materially affect game x AI adoption

## Out Of Scope

- a generic foundation model release with no game or interactive-media connection
- a routine game launch with no meaningful AI component
- generic "AI is changing everything" commentary without concrete game relevance
- unverified rumor threads or speculative leaks
- creator chatter that repeats a story without adding evidence

## Borderline Cases

### Include

- a major model company launches an interactive character experience framed as entertainment or game-like product direction
- a game studio adopts AI localization or content tooling at production scale
- a platform like Unity, Unreal, Roblox, or Steam introduces AI-related creator or runtime capabilities

### Usually Exclude

- a model benchmark improvement unless it is directly tied to a game or interactive deployment
- a new game trailer that only uses "AI" as vague marketing language
- social posts claiming a studio is "all-in on AI" without an official source or credible reporting

## Quick Decision Test

Ask these questions:

1. Would a game-industry operator care about this because it changes how games are built, run, or monetized?
2. Would an AI-industry watcher care because it shows AI moving into games or interactive entertainment?
3. Is the connection explicit enough to survive without speculative interpretation?

If the answer to question 3 is no, drop or down-rank the item.
