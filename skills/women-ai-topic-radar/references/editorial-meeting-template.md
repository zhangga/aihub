# Editorial Meeting Template

Use this template when the user wants a version that feels like an internal topic-meeting handout for a public-account or Xiaohongshu team.

Write in concise, natural Simplified Chinese. Sound like an editor or content strategist preparing materials for a planning meeting.

Before returning the answer, save the report under the current working directory in `docs/women-ai-topic-radar/` using a relative path.

## Header

Start with:

```md
# AI 热点选题会材料

汇报周期：<absolute date range>
适用账号：泛女性成长 / 家庭 / 亲子教育相关内容号
覆盖平台：小红书、微博、微信公众号、X/Twitter
结论先看：<1-2 sentences>
```

`结论先看` should answer:

- what is most worth writing now
- what is hot but not worth following

## Section Order

Always use this order:

1. `## 本轮结论`
2. `## 热点拆解`
3. `## 选题优先级`
4. `## 平台打法建议`
5. `## 风险与避坑`

## 本轮结论

Use three flat lists:

### 值得马上做

List 2 to 4 topics. For each one, include:

- topic name
- one-line reason
- recommended format such as `公众号深写` or `小红书轻表达`

### 可以观察

List 2 to 3 topics that may grow but are not ready yet.

### 不建议跟

List 1 to 3 topics that are mismatched, overdone, or too risky.

## 热点拆解

Use one block per strong topic:

```md
### <topic headline>

热点状态：<爆发 / 升温 / 观察>

这件事是什么：
<2-3 sentences>

为什么会被讨论：
<1-2 sentences grounded in social reaction>

对我们有什么内容价值：
- <value 1>
- <value 2>

建议切口：
- <angle 1>
- <angle 2>
- <angle 3>

不建议怎么写：
- <pitfall 1>
- <pitfall 2>

更适合的发布端：
<公众号 / 小红书 / 双端都可>
```

Rules:

- `这件事是什么` stays factual
- `对我们有什么内容价值` must be audience-specific
- `不建议怎么写` should stop weak or clickbait framing before it happens

## 选题优先级

Summarize the strongest topics in a simple table:

```md
| 题目 | 优先级 | 推荐平台 | 推荐原因 |
|---|---|---|---|
```

Use `S / A / B / C`.

## 平台打法建议

Split this section into two parts:

### 公众号

Describe:

- whether the topic should be trend analysis, opinion, or practical guide
- what emotional tone fits best
- what title style is likely to work

### 小红书

Describe:

- whether the topic should be note, carousel, or short video
- whether the expression should be practical, personal, or interpretive
- what kind of hook fits

## 风险与避坑

Close with a short checklist such as:

- avoid treating rumors as settled facts
- avoid turning parenting topics into panic content
- avoid promising life transformation from immature tools
- avoid writing for "all women" as if the audience were homogeneous

## Tone Notes

Prefer:

- decisive wording
- practical editorial judgment
- short sentences
- clear `do / do not do` guidance

Avoid:

- generic trend summaries
- academic language
- exaggerated emotional manipulation
- abstract AI jargon without creator translation
