# Template Skill

Use this directory as a minimal starting point for a new local skill.

Before filling this template out, use the `skill-creator` workflow to shape the skill properly. This template is the file-level starter, not a replacement for the broader skill design guidance.

## Workflow

1. Use `skill-creator` to define the skill scope, trigger language, and structure.
2. Rename the frontmatter fields.
3. Replace the placeholder purpose with the real behavior.
4. Remove sections that do not help the target workflow.

## Guidance

- Keep the scope narrow.
- Put trigger information in the description.
- Add references or scripts only when repeated work justifies them.
- Treat this file as a starter template after `skill-creator`, not as the full design process.
