# Report Template

Use this Markdown structure for the final output. Write in Simplified Chinese unless the user asks for another language.

Before returning the answer, save the report under the current working directory in `docs/women-ai-topic-radar/` using a relative path.

## Header

Start with:

```md
# AI 选题雷达：女性成长创作者版

时间窗：<absolute date range>
平台范围：小红书、微博、微信公众号、X/Twitter
核心受众：泛女性成长类创作者
```

If the time window or platform coverage was widened or narrowed, say so immediately below the header.

## Section Order

Always use this order:

1. `## 三日热点总览`
2. `## 重点选题卡`
3. `## 趋势判断`
4. `## 本轮选题建议`

## 三日热点总览

Write 5 to 8 bullets.

Each bullet should include:

- the topic in one short phrase
- why it is hot now
- one sentence on why it matters for women-focused growth creators

This section should be fast to scan.

## 重点选题卡

Use one block per shortlisted topic:

```md
### <topic headline>

热度判断：<high / medium / watch>

发生了什么：
<2-3 sentences>

为什么会火：
<1-2 sentences>

与女性成长内容的连接点：
<1-3 bullets or short sentences>

可写角度：
- <angle 1>
- <angle 2>
- <angle 3>

公众号切入：
<one concrete article direction or title style>

小红书切入：
<one concrete note/carousel/video direction or title style>

风险提示：
<what to verify, what to avoid overstating, or why the topic may not fit>

推荐优先级：<S / A / B / C>
```

Rules:

- keep `发生了什么` factual
- keep `为什么会火` tied to real discussion signals
- make the creator angle explicit instead of assuming it is obvious
- add risk notes even for strong topics

## 趋势判断

Summarize the bigger pattern behind the last 3 days.

Good examples:

- discussion is shifting from model capability to everyday life impact
- audiences care more about "does this reduce pressure?" than technical performance
- creator interest is splitting between practical tutorials and emotional interpretation

Prefer 3 to 5 bullets or short paragraphs.

## 本轮选题建议

Always close with three lists:

### 值得立刻跟进

List the 3 strongest topics with one-sentence reasoning each.

### 值得观察

List 2 to 3 topics that may become stronger soon but are not yet ready.

### 不建议跟进

List 1 to 3 topics that look hot but are weak fits, overdone, or risky for the audience.
