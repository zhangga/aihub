# Xiaohongshu Meeting Template

Use this template when the user wants a faster, more platform-native planning memo for Xiaohongshu operations.

Write in concise, natural Simplified Chinese. Keep it short, direct, and easy to use in a content meeting.

Before returning the answer, save the report under the current working directory in `docs/women-ai-topic-radar/` using a relative path.

## Header

Start with:

```md
# 小红书 AI 选题会速览

观察周期：<absolute date range>
目标账号：<creator type>
一句话判断：<what is most worth posting now>
```

## Section Order

Always use this order:

1. `## 这轮最值得发什么`
2. `## 热点拆成什么内容`
3. `## 不建议发什么`
4. `## 发布建议`

## 这轮最值得发什么

List 3 to 5 topics.

Use this format:

```md
- <topic>
热度：<high / medium / watch>
适合发：<note / carousel / short video>
一句话理由：<why it fits Xiaohongshu>
```

## 热点拆成什么内容

Use one block per strong topic:

```md
### <topic headline>

用户为什么会点：
<1-2 short sentences>

建议做法：
- <content angle 1>
- <content angle 2>
- <content angle 3>

建议标题方向：
- <title style 1>
- <title style 2>

表达提醒：
- <what tone to use>
- <what to avoid>
```

Prefer:

- practical hooks
- personal framing
- saveable content
- emotional clarity

Avoid:

- heavy industry jargon
- long policy exposition
- abstract macro analysis without life relevance

## 不建议发什么

List 2 to 4 weak-fit topics and explain why they are poor Xiaohongshu material.

## 发布建议

Always close with:

- `最推荐的 1 条`
- `更适合做图文还是视频`
- `起号型表达还是成熟账号表达`
- `有没有争议风险`
