# Source Map

Use this file to seed source discovery. Treat it as a starting map, not an exhaustive list.

## Primary Source Types

- official company blogs
- product announcement pages
- developer documentation and release notes
- conference or keynote posts
- press releases
- executive interviews in credible publications
- established trade media

## Game-Side Sources To Check

- game engine and platform companies
  - Unity
  - Epic Games and Unreal Engine
  - Roblox
  - Steam platform updates when relevant
- game publishers and studios
  - EA
  - Ubisoft
  - Tencent Games
  - NetEase Games
  - Microsoft Gaming / Xbox
  - Sony Interactive Entertainment
- game-industry trade media
  - Game Developer
  - GamesBeat
  - IGN business or industry coverage
  - PC Gamer when the item is industry-relevant

## AI-Side Sources To Check

- major AI labs and vendors
  - OpenAI
  - Anthropic
  - Google DeepMind
  - NVIDIA
  - Meta AI
  - Runway
- AI startup and tooling sources when the product clearly touches games or interactive media
- AI trade and business media covering product launches, funding, or partnerships

## Source Balancing Guidance

Try to cover both of these buckets in each report when signal exists:

- `Game Companies Using AI`
- `AI Companies Building Game Products`

If one bucket is empty, say so rather than forcing weak items into it.

## X As Default Heat Source

Use X or Twitter as the default platform for heat checks unless the user asks for a different platform.

What to look for on X:

- breakout discussion around a formal announcement
- reactions from developers, artists, studio operators, or engine communities
- strong disagreement between audiences
- repeated sharing by influential accounts or industry-adjacent voices

## Secondary Social Sources

Use these after checking X, or when they add audience color that X does not:

- Reddit
- Weibo
- YouTube creator commentary
- public creator or developer communities visible on the open web

## What Social Validation Should Add

- whether the story is getting unusual attention on X
- whether practitioners see the announcement as meaningful or hype-driven
- whether the reaction differs by audience, such as players versus developers
- whether X is surfacing a concern before formal reporting catches up

## What Social Validation Should Not Do

- serve as the only basis for a core report item
- replace formal sourcing for factual claims
- turn weak rumor into a headline
